Privet 1.3 (Приветик)
==============================================
MS Agent общается с Вами и дает советы. 
В программный код заведены имена пользователей,
работающих на компьютере и/или в локальной сети.
В случае совпадения имени текущего пользователя 
с заведенным именем, MS Agent обращается 
к пользователю по имени.
