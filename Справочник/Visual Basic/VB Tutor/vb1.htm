Сегодня мы будем рассматривать четыре первых инструмента, это: TextBox; Picture; Label; Frame. Вот так они выглядят:
Мы начнём по-порядку.
[TextBox] Текстовое поле служит для ввода информации. Как и у любого объекта у TextBox'а есть свойства. Закинь на форму TextBox. В нём автоматом появиться текст Text1 - этот текст изменяется в свойстве Text (Меню Свойств).
Так же имеется свойство Name - это имя объекта, оно может быть написано только латинскими буквами и/или цифрами.
MultiLine: true(истина) - многострочный режим ввода текста; false(ложь) - однстрочный режим ввода текста.
MaxLenght: ограничитель выводимых символов, т. е. если это свойство равно 5, то больше пяти символов в текстовое поле будет ввести нельзя.
PasswordChar: маска выводимых символов, т. е. если если он будет равен '*', то все введёные символы будут прятаться под звёздочками и так же идентично с остальными символами.
ScrollBars: это свойство нужно для установки скроллбаров (полос прокрутки).

[Picture] Полное название [PictureBox], но мы не будем сильно напрягаться. Основные свойства:
Picture: собственно картинка.
AutoSize: авторазмер, т. е. размер компонента будет соответствовать размеру вставленной в неё картинки.
Appearance: режим компонента - 3D (вмятый) или Flat (плоский).
BorderStyle: Fixed Single (вмятый) или None (нет никакого, т. е. плоский, но не как Flat).

[Label] Метка. В программе, где есть метка нельзя её менять. Свойства:
Caption: собственно текст на метке.
AutoSyze: авторазмер под написанный текст.
BackStyle: режим прозрачности или наоборот.
Font: шрифт, его размер и свойства.
ForeColor: цвет шрифта.

[Frame] Фрэйм нужен в основном для оформления. Свойства:
Caption: надпись на фрейме.

Вот мы и рассмотрели четыре первых компонента VB6. Тебе надо усвоить главную вещь - существуют свойства индивидуальные и есть такие свойства, которые присуствуют почти у каждого компонента, это:
Enabled: True (включено) или False (отключено).
Visible: True (виден) или False(не виден).
Name: имя.
Width: ширина.
Height: высота.
Left: положение, относительно оси X.
Top: положение, относительно оси Y.
В VB6 координаты начинаются с верхнего левого угла формы (0,0), а координаты и размеры выражаются в Твипах (Twip); ты, конечно, можешь поменять эту единицу измерения на любую доступную, но учить этому я тебя не буду, т. к. работать мы будем только с Твипами.

Сейчас мы будем рассматривать ещё четыре немаловажных объекта, это: CommandButton; CheckBox; OptionButton; Timer. Вот так они выглядят:
По традиции начнём по-порядку.
[CommandButton] Очень важный объект для создания приложений, простое название - кнопка. Свойства:
Caption: текст на кнопке.
Style: стандартная кнопка и графическая.
Picture: картинка на кнопке, работает только в графическом режиме.
DownPicture: картинка на нажатой кнопке, так же работает только в графическом режиме.
BackColor: цвет кнопки, и опять, работает только в графическом режиме.

[CheckBox] Ну, и с этим ты встречался, но раньше ты называл это флажок, а теперь придётся Чекбокс, т. к. флажок - это галочка внутри, а не сам объект. Свойства:
Alignment: свойство изменения отношения текста к Чекбоксу.
Value: это, пожалуй, главное свойство Чекбокса - установка флажка Unchecked (не выбран), Checked (выбран), Grayed ("на половину").

[OptionButton] Радиокнопка - выбрать можно только одну из множества или одну из множества во фрейме. Свойства:
Value: может принимать значения True (точка есть) или False (точки нет). В коде можно записать 1 (точка есть) или 0 (точки нет). [Timer] Таймер - служит для установки переменных во времени. Свойства:
Interval: интервал времени в миллисекундах.
