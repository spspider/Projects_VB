Создания хелпа с помощью WorkShopa. Сноски:

Ну как я уж и говорил, что нужен будет Microsoft Word(ил любой RTF редактор). У меня ща стоит Word 2000, в 98 такая же хрень.
Ну начнем.
В начале надо создать файл с типом RTF, потом его откомпилировать. Начнем с создания, открой Word. Теперь можешь туды пихать рисунки, заголовки, и т.д. и т.п. Но вот например чтоб ссылку сделать, ил еще чо нить в этом роде нужны сноски. Для их добавления нажми на Вставка>>Сноска. Открывается окно, в нем жми на переключатель Другая и в текстовом поле впиши имя(ил фамилию :-)) сноски.


Потом у тебя окно Worda поделиться на 2 части(1 - ая это основной документ, 2 - ая это сноски). Если не поделиться то жми на Вид>>Сноски
Вот основные типы сносок:

$ - Используется для отображения списка тем на закладке «Поиск», в хронологии, закладках.
K - Слова написанные после этой сноски будут состовлять список "Предметный указатель"
# - Это для создание ссылок
+ - Темы отмеченные плюсом связываются в группу по которой можно перемещаться кнопками >> и <<.

P.S. Для разрыва страницы, надо нажать на CTRL+Enter, и тогда появится полоска разделяющая страницу(т.е. это в справке как новый лист будет)

ШАГ 133

Кнопка Options... Компиляция RTF в HLP.:

Для начала открой WorkShop, нажми на File>>New у тебя откроется окно New, в нем из списка выбери Help Project, и жми на OK. Потом откроется окно сохранения файлов, короче сохрани файл на рабочий стол. После сохранения, появляется основное окно:

Теперь надо все это дело настроить, и откомпилировать наш файл из RTF в HLP. Начнем с кнопки Options:

Кнопка Options вкладка General:

Default Topic - сюда ты можешь вписать имя закладки(но необязательно), например если ты впишишь R то у тебя будет грузиться страница с закладкой R(т.е. 2ая страница), ну короче сюда пока ни чо не вписывай.
Help title - вписанный сюда текст будет заголовком у справки.
Display this text in the Version dialog box - вписанный сюды текст будет отоброжаться ( в откомпилированной справки) когда чел входит в Справка>>Версия
If users paste or print help, display - вписанный сюды текст будет выводиться в конце странице при печати, ил когда чел скопирует текст(ил символ), и вставит его.(например, если чел скопирует слово из справки, и вставит его в блокнот, то к вставленному слову прибовляется(на след. строке) еще и текст вписанный в это текстовое поле)

Кнопка Options вкладка Compression:

Сдесь можно поставить коэфицент сжатия, короче всегда ставь на Maximum(чтоб меньше весило)

Кнопка Options вкладка Files:


Help File - имя откомпилированного файла.
Log File - сюды можешь ни чо не вписывать, а вобще это имя log файла, т.е. после компиляции появляется текст описывающий процесс компиляции.
Rich Text Format files - сдесь выбери файл который мы делали в предыдущем шаге, т.е. RTF
Contents file - сдесь надо выбрать файл со списком тем, но об этом в следующих шагах.
Tmp folder - сдесь надо выбрать папку для хранения временных файлов, т.е. файлов создовающихся в процессе компиляции.
P.S. Теперь можешь откомпилировать справку, для этого нажми на кнопку Save and Compile, ил File>>Save, потом File>>Compile
Все файлы ты можешь скачать Здесь

ШАГ 134

Кнопка Files:

Эта кнопка нужна для добавления/удаления дополнительных RTF файлов.
Вот например, ты можешь создать новый RTF файл, и сделать в нем закладку(например #H(тока не забудь добавить его(кнопка Add(можно и удалить кнопкаRemove)))), и в старом(который мы предыдущем шаге делали) указать ссылку на #H, и у тебя будет все это работать.
Короче ссылки на закладки в другие RTF файлы делаются также как и обычные ссылки, т.к. после компиляции все в одну кучу сваливается.
Короче если ты ни фига не понял, то качай пример(внизу).
Можно выбрать папку с RTF, и тогда добавються все RTF файлы которые в этой папки(кнопка Folder)

ШАГ 136

Кнопка Windows:

Нажми на кнорку Windows, у тебя окно Create a window, в текст. поле впиши - "New", это и будет новое окно.

Жми на ОК.
Вот свойства:
Кнопка Windows вкладка General:


Windows type - имя нового окна(если будет несколько имен, то для каждого выбранного в этом списке окна, будут устанавливаться(тобой) отдельные свойства)
Title bar text - Заголовок окна
Window attribute - Атрибуты окна связанные с его размещением на экране:
auto-size height – размер окна автоматически подбирается по текст.
maximize window – окно разворачивается на весь экран.
keep help window on top – окно не перекрывается другими окнами.

Кнопка Windows вкладка Position:

Window type – имя нового окна
Adjust for user’s screen resolution – Автоматический подбор размеров под разрешение экрана
AutoSizer – с помощью этой кнопки можно подобрать размеры окна. Короче, жмешь на кнопку, открывается окно, устанавливаешь размеры этого окна, и жмешь на OK. Новое окно будет таких - же размеров как и это.
Кнопка Windows вкладка Button:

Сдесь можно установить кнопки, которые отброжаться в новом окне будут:

Contents (Разделы) – список тем справки (необходим CNT файл).
Index (Указатель) – поиск по ключевым словам.
Find (Поиск) – поиск.
Print (Печать) – печать темы.
Back (Назад) – вернуться на тему назад.
Browse (<<,>>) – последовательное перемещение по темам.
Option (Параметры) – параметры отображения.

Кнопка Windows вкладка Color:

Установка цвета окон, об этом потом.

Делаем файл со списком тем:

Открой WorkShop, нажми на File>New, в окне New

Жми на OK, и перед тобой откроется окно:

В текст. поле Default title впиши заголовок справки.
Нажми на кнопку Add Below, кликни по переключателю Heading, и в поле Title впиши заголовок.

Тепрерь опять нажми на Add Below, кликни по переключателю Heading, и в поле Title впиши заголовок. Жми на OK. Теперь выдели созданный список, и кликне по кнопке Move Right

Теперь сделаем ссылку, Add Below, кликни по переключателю Topic:
В поле Title - впиши заголовок
В поле Topic ID - впиши имя закладки(например если закладка #R то впиши просто R)
Жми на OK, сохрани(File>Save As) проект в той же папки где и остальные файлы справки. ОК? Теперь войди в WorkShop(Help Project(хотя нет, лучше откроЙ СТАРЫЙ файл *.hpj)). Нажми на кнопку Options вкладка Files, и в поле Contents File выбери файл со списком(на кнопку Browse кликни. OK?). Теперь откомпилируй проект, и любуйся!

Вызываем файл справки из VB:

Вобще - то у меня в разделе(Статьи) есть Статья, специально на эту тему, но я в краце скажу про это:
Войди в VB(Standart EXE), нажми на Проект(Project)>>Project1 Свойства(имя_проекта Properties)(у меня VB на половину русский, если у тебя English, то качай с моего сайта руссификатор.).
В окне свойств напротив поля имя нажми на кнопку ..., и выбери там файл справки. Теперь код:

'При нажатии на кнопку вызываем файл справки(App.HelpFiles)
