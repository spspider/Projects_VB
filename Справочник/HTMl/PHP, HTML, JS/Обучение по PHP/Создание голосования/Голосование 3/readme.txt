Baltic Poll - это голосование, которое было написано для проекта Scripting Project (www.script.ee) Если будут какие-либо проблемы с голосованием, то можете обращаться сюда: haze@script.ee

Описание:
1.Поддержка SSI 
2.header, footer 
3.Полностью и легко изменяемый вывод результатов 
4.Изменение шаблонов из Админ. интерфейса 
5.Возможность добавить/редактировать/аннулировать/удалить голосование из админ. интерфейса 
6.Использование словаря, то есть голосование можно будет перевести на другой язык, прямо из админ. интерфейса... 
7.Защита от повторного голосования (можно отключить) 
8.Использование одной картинки для всех вариантов ответа или разных для каждого... 
9.Возможность выводить результаты все или с какой-либо и по какую-либо дату... 
10.В админ. интерфейсе возможность полного анализа результатов, включая общие результаты, результаты с какого-либо и по какое-либо число. 
11.Голосовать можно будет либо переключателями (radio), либо чекбоксами (чтобы сразу голосовать за несколько вариантов ответа). 
12.При первом запуске запускается скрипт, который позволяет отредактировать все пути и настройки через веб интерфейс. Конечно, потом, все это можно будет поменять из админ. интерфейса. 
13.Поддержка cookies 
14.Обработка нестандартных ситуаций и ошибок, не нарушая, работу программы 
15.Автоматическая замена голосования со старого на новое (в файле) через админ. интерфейс.


Установка:
1.Измените верхнюю строчку в файлах balticpoll.cgi и admin.cgi (#!/usr/bin/perl) на правильный путь к Перлу.
2.Все, что лежит в папке cgi-bin запишите в аналогичную папку на сервере, а то, что лежит в public_html в корневую директорию сервера.
3.Установите для директории, где лежит скрипт CHMOD равным 755, 
CHMOD для файлов с расширением .cgi - 755
CHMOD для всех остальных файлов (в CGI-BIN директории) - 666
4.Запустите admin.cgi
